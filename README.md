# meuslinks · db.

Página de links pessoal de Daniella Bastos — estilo linktree, feita à mão com HTML e CSS puro.

## Tech Stack

- HTML5 semântico
- CSS3 (custom properties, flexbox, animações)
- Google Fonts (Bricolage Grotesque, DM Sans, Instrument Serif, JetBrains Mono)
- Sem frameworks, sem dependências

## Estrutura

```
links/
├── index.html   # markup semântico
└── style.css    # design tokens, layout e componentes
```

## Como rodar localmente

Basta abrir o `index.html` no navegador — não há build, servidor ou instalação necessários.

```bash
git clone https://github.com/danibsoares/meuslinks.github.io.git
cd meuslinks.github.io
# abra index.html no navegador
```
